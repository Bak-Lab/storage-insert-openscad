Quick Start

Get running in 30 seconds

Requires Python 3.10+ and OpenSCAD (optional if you only want .scad files).

Ubuntusudo apt install openscad
Archsudo pacman -S openscad
macOSbrew install openscad

# Default storage layout, 300×400×50 mm
python generate.py --width 300 --length 400 --depth 50

# Uniform 4×3 grid
python generate.py --width 300 --length 400 --depth 50 --layout grid 4 3

# Custom layout from JSON
python generate.py --width 300 --length 400 --depth 50 --layout custom examples/custom_layout.json

# Just .scad, no OpenSCAD needed
python generate.py --width 300 --length 400 --depth 50 --scad-only

Layouts

Layout Modes

storage

Classic insert — large + small on top row, three equal cells on bottom. The default.

grid COLS ROWS

Uniform grid of equal-sized compartments. Simple and fast.

custom PATH

JSON file with a 2D array of relative proportions per row. Full control.


Custom JSON example:

{
  "layout": [
    [2, 1],
    [1, 1.5, 1],
    [1, 1]
  ]
}
